SR-14C бор алмазний турбінний (141/018)

Алмазні бори для турбінного наконечника, що застосовуються в стоматології. Для виготовлення борів використовується природний алмазний порошок, завдяки якому робоча частина бора має дуже високу твердість та відмінні фізичні та хімічні особливості.

Переваги

  • Робоча частина виготовлена з нержавіючої сталі та покрита дрібним кристалом алмазу.
  • Використовується як обертовий шліфувальний інструмент для твердих тканин, таких як зуб і кістки.
  • Для турбінного наконечника.
  • Можна використовувати для шліфування металу, пластику, фарфору та інших твердих матеріалів.

 

Застосування

  • Продукт використовується для препарування зубів у ротовій порожнині під час лікування.
